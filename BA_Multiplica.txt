PROGRAMACION

Definicion de desarrollo web: especialidad encargada de la informatica encargada de construir y mantener web

full stack = combinacion de front end y back end 

internet = red de redes que contiene materiales, fuentes de inf y almacenam y diversos nodos con dist jerarquias 

servidor = computadora que presta serv e inf a partir de un determ software y funciona como fuente para ser usada por cualquier usuario 

tcp/ip = 2 protocolos de internet que determ como se realiza la transmision de datos 

estos protocolos trabajan en conjunto que se encargan de identificar y establecer la mejor ruta para el origen (el cliente) y el destino (el servidor). A su vez da seguridad porque envia inf de forma fragmentada (paq de datos) y la unifican en el destino

IP PRIV
identifica equipos dentro de una red privada. gral redes que no sean la propia internet.

DNS 
POSEE BASE DE DATOS DE DIRECCIONES IP Y NOMBRES DE DOMINIO DE INTERNET 

si escribimos www.google.com (nombre de dominio) podemos obtener la direccion ip y pedir inf al servidor correcto
